# #14 My Menu Roulette

1 Tool / 1 Action: Decide your meal from a personalized list with one tap.

<img src="icon.png" width="100" alt="Icon">

## Core Philosophy
- **1 Tool / 1 Action**: Focuses solely on randomly selecting one item from a user-defined list. No complex features.
- **Performance**: Browser-only application, designed to run instantly on low-spec PCs and mobile devices.
- **Privacy**: No login, no tracking, no ads. Your menu list is saved only in your browser's LocalStorage.

## Design System
- **Colors**: Background `#fff` / Main `#333` / Accent `#eee` (Input Area `#fafafa`).
- **Radius**: Mandatory `15px` radius applied to all elements (buttons, input fields).
- **Style**: iOS/Android standard sans-serif font. Utilizes bold text for a functional, "tool-like" UI.

## Strict Protocol
- **Originality**: The structure and logic of the original HTML are strictly preserved when making any modifications.
- **Analysis**: Text analysis is prioritized to determine UI composition.
- **Language**: English only for README.md.
